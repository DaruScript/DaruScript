## Build Instructions
Clone the GitHub Repository:
```bash
git clone https://github.com/DaruScript/DaruScript
```

Build the project:
```bash
cargo build
```

Run the project:
```bash
cargo run
```

## Usage Instructions
```bash
Usage: daru_script.exe [OPTIONS]

Options:
-p, --printast
-h, --help      Print help
-V, --version   Print version
```
